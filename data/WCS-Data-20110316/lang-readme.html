WCS DATA ARCHIVES: lang.txt README

The `lang.txt' file contains information on the WCS language names.

Each line of the file has 8 tab-separated fields. Empty fields have "*" asterisk. Only the first 4 fields are described here (the others were for WCS bookkeeping).

field number

field type

field value

1. WCS Language Number 1..110
2. WCS Language Name Hex-escaped Mac encoding
3. WCS Language Geographic Location Name
4. Field Worker Name
