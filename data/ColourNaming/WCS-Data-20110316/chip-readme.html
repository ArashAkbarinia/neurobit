WCS DATA ARCHIVES: chip.txt README

The `chip.txt' file contains WCS grid mappings.

Each line of the file has 4 tab-separated fields.

field number

field type

field value

1. WCS Chip Number 1..330
2. WCS Grid Row A..J
3. WCS Grid Column 0..40
4. Concatenation of fields 2 and 3
