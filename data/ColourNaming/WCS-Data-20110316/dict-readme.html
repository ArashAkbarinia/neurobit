WCS DATA ARCHIVES: dict.txt README

The `dict.txt' file contains WCS dictionary data for each of the 110 languages. Note that this file uses UTF-8 encoding form.

Each line of the file has 4 tab-separated fields.

field number

field type

field value

1. WCS Language Number 1..110
2. Term Number integer, sequential numbering of terms per language
3. Term fieldworker's transcription of the color term
4. Term Abbreviation per language, a unique abbreviation of the term
