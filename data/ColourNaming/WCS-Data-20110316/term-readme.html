WCS DATA ARCHIVES: term.txt README

The `term.txt' file contains data for the WCS naming task (chip responses).

Each line of the file has 4 tab-separated fields.

field number

field type

field value

1. WCS Language Number 1..110
2. WCS Speaker Number Integer, total varies per language
3. WCS Chip Number 1..330 (see `chip.txt')
4. Term Abbreviation Per language
